# Mesto Russia

Одностраничный сайт посвященный путешествиям по России. В основной функционал входит адаптивная верстка с построением сетки Grid Layout и модальное окно.

#### Использовались следующие технологии:

* BEM

* Flexbox

* Grid Layout

* Positioning

* Pseudo-classes

* Media Queries
